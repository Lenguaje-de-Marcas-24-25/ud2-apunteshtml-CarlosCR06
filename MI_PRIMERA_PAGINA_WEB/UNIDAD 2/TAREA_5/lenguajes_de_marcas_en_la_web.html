Respuestas a las preguntas sobre HTML y XHTML

1. ¿Qué es el lenguaje HTML?

HTML (Hypertext Markup Language) es el lenguaje de marcado utilizado para estructurar el contenido en la web. Define la estructura de las páginas web mediante el uso de etiquetas como encabezados, párrafos, listas, enlaces, imágenes, entre otros.

2. ¿Cuál es la relación de Tim Berners-Lee con HTML?

Tim Berners-Lee es el inventor de HTML. Lo desarrolló en 1991 como parte de la creación de la World Wide Web (WWW), que permitió la publicación de información en la web de manera accesible para todos.

3. ¿Cuáles son las distintas versiones del lenguaje HTML?

Las versiones principales de HTML son:

  • HTML 1.0 (1991)
  • HTML 2.0 (1995)
  • HTML 3.2 (1997)
  • HTML 4.01 (1999)
  • XHTML 1.0 (2000)
  • HTML5 (2014, la versión más reciente)

4. ¿Qué es la W3C?

La W3C (World Wide Web Consortium) es la organización encargada de desarrollar estándares y pautas internacionales para garantizar el crecimiento a largo plazo de la web. Define especificaciones para tecnologías como HTML, CSS, XML, entre otras.

5. ¿Qué es el WHATWG?

El WHATWG (Web Hypertext Application Technology Working Group) es un grupo de trabajo compuesto por empresas como Apple, Mozilla, y Google, que se enfoca en el desarrollo y evolución de HTML y sus especificaciones.

6. ¿Qué es un Living Standard?

Un Living Standard es un estándar web que está en constante actualización. A diferencia de otros estándares, no tiene una versión "final", sino que se actualiza regularmente para adaptarse a nuevas tecnologías y necesidades.

7. ¿Qué es el lenguaje XHTML?

XHTML (Extensible Hypertext Markup Language) es una versión más estricta de HTML que se basa en XML. Fue diseñado para mejorar la interoperabilidad entre los navegadores y otros dispositivos, forzando una sintaxis más rigurosa.

8. ¿Cuáles son las distintas versiones del lenguaje XHTML?

Las principales versiones de XHTML son:

  • XHTML 1.0 (la más utilizada, lanzada en 2000)
  • XHTML 1.1 (una versión más estricta)
  • XHTML 2.0 (abandonada antes de ser completada)

9. ¿Cuáles son las diferencias sintácticas y estructurales entre XHTML y HTML?

Las principales diferencias son:

  • XHTML requiere que todas las etiquetas estén cerradas correctamente (por ejemplo, <br /> en lugar de <br>).
  • XHTML es más estricto en cuanto a la sintaxis: por ejemplo, los atributos deben ir entre comillas y las etiquetas deben ser siempre en minúsculas.
  • HTML es más permisivo con la sintaxis, permitiendo que algunas etiquetas se omitan o no se cierren adecuadamente.
